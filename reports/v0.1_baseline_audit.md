### Cosa NON è stato testato (gap analysis)

| Area | Priorità | Impatto se non fatto |
|------|----------|---------------------|
| **Pipeline C** (Markdown + shallow filter) | Alta | Il paper non può rispondere a "Markdown peggiora per chunk superficiali o per Markdown in sé?" |
| **Bootstrap / CI** | Alta | Non abbiamo stime di incertezza sui delta; non possiamo dire se Δ=-0.18 è significativo |
| **Permutation test** | Alta | Non abbiamo p-value; non possiamo falsificare H0 |
| **Secondo embedding model** | Media | Lo shallow chunk effect potrebbe essere specifico di all-MiniLM-L6-v2 |
| **LLM-as-judge** | Media | La scoring proxy (fuzzy overlap) può nascondere equivalenze semantiche |
| **Human audit** | Media | Nessun ground truth umano per validare il scoring |
| **Documenti oscuri/sintetici** | Media | Parametric memory confound non controllato |
| **Famiglia modelli controllata** | Bassa | Il gradient model-size × pipeline è confondato da provider/API |
| **Ground truth table parsing** | Bassa | Table extraction non ha parser dedicato |

### Risultati principali (riassunto)

| Pipeline | Mean score (/5.0) |
|----------|:-----------------:|
| A — Raw | 2.50 |
| B — Markdown | 2.56 |
| Δ (B−A) | +0.06 (+1.2%) |

Gradient 4-modelli:

| Model | Size | A (Raw) | B (MD) | Δ |
|-------|------|:-------:|:------:|:-:|
| Qwen 0.8B | 800M | 2.50 | 2.56 | +0.06 |
| Nemotron 3 | ~3B | 2.46 | 2.28 | -0.18 |
| DeepSeek V4 | — | 2.98 | 2.76 | -0.22 |
| Gemma 4 26B | 26B | 3.12 | 2.86 | -0.26 |

Shallow chunk finding: 38/150 (25.3%) chunk recuperati da Pipeline B hanno informative_len ≤200 caratteri vs 0% in Pipeline A.

### Limitazioni bloccanti

1. **Provider confound**: ogni modello usa provider, API, tokenizer diversi — il gradient non è attribuibile a model size.
2. **Pipeline B è una sola implementazione Markdown**: non generalizzabile ad altri compilatori o chunker.
3. **Scoring proxy**: fuzzy overlap è una metrica grossolana che penalizza risposte parafrasate.
4. **Nessuna inferenza statistica**: Δ riportati senza CI o p-value.
5. **Nessuna Pipeline C**: il meccanismo proposto (shallow chunk) non è stato testato con ablation.
